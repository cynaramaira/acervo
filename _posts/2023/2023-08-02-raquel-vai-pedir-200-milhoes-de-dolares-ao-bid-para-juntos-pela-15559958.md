<p>A governadora Raquel Lyra (PSDB) quer recursos internacionais em seu projeto para a seguran&ccedil;a p&uacute;blica, anunciado esta semana.</p>
<p>A gest&atilde;o tucana enviou um projeto de lei para autorizar que o Governo do Estado destaque, dos pedidos de empr&eacute;stimo do Banco Interamericano de Desenvolvimento (BID) uma quantia para o projeto "Juntos pela Seguran&ccedil;a".</p>
<p>"Do valor total, o Poder Executivo poder&aacute; contratar opera&ccedil;&atilde;o de cr&eacute;dito junto ao Banco Interamericano de Desenvolvimento - BID no montante de at&eacute; US$ 200 milh&otilde;es (duzentos milh&otilde;es de d&oacute;lares dos Estados Unidos da Am&eacute;rica) para o Projeto Juntos pela Seguran&ccedil;a", diz o texto do projeto.</p>
<p>A governadora solicitou que os deputados estaduais aprovem o projeto em regime de urg&ecirc;ncia.</p>
